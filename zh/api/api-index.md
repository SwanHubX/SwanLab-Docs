## 命令行

- [swanlab ping](./cli-swanlab-ping.md): 连通性与环境诊断
- [swanlab api](./cli-swanlab-api.md): 通过命令行操作云端工作空间/项目/实验/用户资源
- [swanlab watch](./cli-swanlab-watch.md): 启动离线实验看板
- [swanlab login](./cli-swanlab-login.md): 登录SwanLab
- [swanlab logout](./cli-swanlab-logout.md): 登出SwanLab
- [swanlab verify](./cli-swanlab-verify.md): 验证登录状态
- [swanlab convert](./cli-swanlab-convert.md): 将其他产品的日志转换为SwanLab项目
- [swanlab sync](./cli-swanlab-sync.md): 将本地日志同步到SwanLab云端/私有化部署端
- [swanlab local](./cli-swanlab-local.md): 将SwanLab设置为本地模式
- [swanlab online](./cli-swanlab-online.md): 将SwanLab设置为云端模式
- [swanlab offline](./cli-swanlab-offline.md): 将SwanLab设置为离线模式
- [swanlab disabled](./cli-swanlab-disabled.md): 将SwanLab设置为禁用模式
